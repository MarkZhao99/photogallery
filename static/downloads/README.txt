Wallpaper Helper for macOS

1. 解压 wallpaper-helper-macos.zip
2. 双击 wallpaper-helper-macos.command
3. 首次运行时，macOS 可能会提示来源确认；请选择继续打开
4. 当系统提示 Terminal 或 Python 需要控制 System Events / 桌面时，请允许
5. 保持终端窗口开启，然后回到展示页即可使用“一键设为当前电脑壁纸”

如果没有自动启动：

- 在终端里进入解压后的文件夹
- 运行：chmod +x wallpaper-helper-macos.command
- 再运行：./wallpaper-helper-macos.command
